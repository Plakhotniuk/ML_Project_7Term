# Проект по курсу: введение в машинное обучение.

## Описание

- Агент в виде машины расположен в реалистичной физической среде и может перемещаться по одномерной дороге в виде ямы. 
- Агенту доступно одно из двух управяющих воздействий: ускорение вперед и назад. 
- Агенту необходимо выбраться из ямы, достигнув флага на одном из ее краев, за минимальное время.

![car](/plots/car.png)

### Цель
Обучить агента особенностям физической среды, чтобы достигать флага за минимальное время.

### Задача машинного обучения
Обучения с подкреплением, Q-learning
